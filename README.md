# LALA contribution

> [서비스 링크](https://lala.harvester.kr)

LALA는 Private 프로젝트이므로 외부 공유가 불가합니다.

때문에 이 `lala-contribution` repo에서 프로젝트에서 제가 기여한 부분에 대한 것만 따로 분리하여 정리해두었습니다.

기여한 부분에 대한 정리는 마크다운 형식으로 하나씩 추가할 예정입니다.
